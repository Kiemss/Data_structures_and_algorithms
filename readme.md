# 数据结构与算法
## 顺序表
### 动态数组

* 特点：内存连续

1. 优点

   * 下标访问（随机访问）时间复杂度是O(1)
   * 尾删、尾插时间复杂度是O(1)
   * 访问元素前后相邻位置的元素非常方便

2. 缺点

   * 非末尾位置增删元素需要进行大量的数据移动

   * 搜索的时间复杂度

     * 无序数组：线性搜索O(n)
     * 有序数组：二分搜索O($\log{n}$)

     数组扩容消耗比较大

3. 注意事项

   * 数组的查找（搜索）和下标访问（随机访问）是不同的概念！
     前者是根据元素找到对应的索引，后者是根据索引给出对应的元素。
