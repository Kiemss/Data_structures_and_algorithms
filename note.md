# 笔记

## 命名规范

* 头插	push_front
  尾插	push_back
  头删	pop_front
  尾删	pop_back
* 检查是否存在            contain
  查找并返回迭代器    find
* 删除第一个指定元素    erase
  删除所有指定元素        remove
* 遍历    display



## 易错点

1. 成员函数在类内声明时若已经给出默认参数，则类外定义时无需给出默认参数

2. 默认参数中不能使用类的非静态成员变量，即使不使用 this 指针也不行。这是因为：

   * 默认参数在编译时确定

   * 非静态成员变量在运行时才存在
